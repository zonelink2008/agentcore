# AgentCore 核心设计理念

> ⚠️ **每次功能设计前必读**

---

## 核心原则

**让普通 Agent 调用专业 Agent 来完成任务**

- 普通 Agent 发布任务，询问用户是否要"高效高质量"
- 用户选择"是"后，AgentCore 智能匹配专业 Agent
- 专业 Agent 执行任务，结果返回给用户

```
人类用户 → 普通 Agent → AgentCore → 专业 Agent → 结果
```

---

## 四大支柱

1. **任务撮合** - 智能匹配专业 Agent
2. **Agent 池** - 各类专业 Agent 入驻
3. **Agent 路由** - 根据任务类型自动匹配
4. **多Agent协作** - 分解复杂任务

---

## 设计检查清单

- [ ] 这个功能是给 Agent 用的还是人类用的？
- [ ] Agent 如何调用专业 Agent？（API 优先）
- [ ] 是否需要人类选择"高质量"模式？
- [ ] 交易/计费是否在 Agent 之间完成？
