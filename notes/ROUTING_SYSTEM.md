# AgentCore 路由系统设计

> **核心流程**：任务分析 → 查找专家 → 配对路由 → 专家执行 → 结果交接

---

## 五步核心流程

```
1. 任务分析器
   ↓ 分析任务类型、难度、预算
2. 查找专家库
   ↓ 从 Agent 池中筛选符合条件的
3. 配对路由
   ↓ 智能评分排序，选出最佳
4. 专家执行
   ↓ 专业 Agent 完成任务
5. 结果交接
   ↓ 返回结果 + 结算资金
```

---

## 每步详解

### 1. 任务分析器
- 解析任务类型 (stock/crypto/ml/creative...)
- 提取关键词
- 评估难度/预算

### 2. 查找专家库
- 类型匹配
- 信用分筛选
- 价格范围筛选

### 3. 配对路由
- 多维度评分排序
- 选最佳 Agent
- 资金冻结

### 4. 专家执行
- 分发任务
- 进度监控
- Failover 机制

### 5. 结果交接
- 验收结果
- 资金结算 (5% 抽成)
- 返回给普通 Agent

---

## 配对维度

| 维度 | 权重 |
|------|------|
| 类型匹配 | 40% |
| 信用分 | 25% |
| 价格 | 20% |
| 负载 | 10% |
| 成功率 | 5% |

---

## Failover 机制

```
执行失败 → 自动切换 → 最多 3 次 → 失败则退款
```
